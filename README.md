该流量监控app只用来交流学习，查看套餐流量，切勿用于他用，否则后果自负。

点击列表套餐，会记录当前使用值，下次刷新将自动进行对比。


点击近期使用，会记录当前使用值，下次刷新将自动进行对比。


点击图片区域将会刷新记录。
